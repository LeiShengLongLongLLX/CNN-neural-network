### 表格说明
1. **Depthwise Conv（type 2）**：这是 MobileNetV1 的核心创新，每个卷积核只处理一个输入通道，`groups=Cin` 。
2. **Pointwise Conv（type 1）**：1×1 卷积，用于组合深度卷积的输出，改变通道数 。
3. **Stride 处理**：下采样（H/W 减半）发生在部分 Depthwise Conv 层（stride=2），Pointwise Conv 的 stride 均为 1 。
4. **池化层**：最后使用 Global Average Pooling 将 7×7 降为 1×1，type=0（FFN）。
5. **全连接层**：输出 1000 类（ImageNet），type=3（GEMM）。

这个表格完整覆盖了 MobileNetV1 的 **28 层结构**（按深度可分离卷积分开计算）

### 格式说明

- **Cin**：输入通道数（Input Channels），原 C
- **H, W**：输出特征图的高度和宽度
- **R, S**：卷积核的高度和宽度
- **type**：
  - 0 = FFN（池化、输入层等）
  - 1 = CONV（普通卷积）
  - 3 = GEMM（全连接矩阵乘）
